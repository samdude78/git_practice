HTML CheatSheet

Basic HTML Tags

Tags Attributes Name Description
<head> Head Contains meta information about your page.For e.g. <title>
<body> Body Elements we use to build a webpage.Only content inside the opening and closing body tags can be displayed on the screen.
<h1>to<h6> Headings There are six different headings.h1 is used for main/largest heading,h2 for subheading,etc. From largest to smallest in size.
<!-- --> Comment leave comments for other developers within your code without affecting the resulting output
<p> Paragraph Defines a paragraph containing a block of plain text
<div> class,id Division A container that divides the page into sections.Useful for grouping elements together.
<a> target,href Anchor Defines a hyperlink.The href attribute specifies the URL of the page the link goes to.
<span> Span Contains short pieces of text.They are used to separate small pieces of content that are on the same line
<em> Emphasizes Emphasizes text making them italic.
<strong> Strong Highlights important text making them bold.
<br> LineBreaks Adds a line break which can be seen in browser.Self closing tag.
<ul> Unordered Lists Outlines individual list items with a bullet point.
<ol> Ordered Lists Outlines individual list items in a numbered way.
<li> List Item Used to describe an item in a list.
<img> src,alt Image Allows you to add an image to a web page.Self closing tag.Has a required attribute called src which must be set to the source or location of image.
<video> src,width,height Video Allows you to display videos
